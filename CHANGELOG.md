# Change Log

All notable changes to the "vscode-nyan-progress" extension will be documented in this file.

## [0.0.1] - 2025-11-04

- Scaffold VS Code extension project structure in TypeScript.
- Add status bar indicator, panel view, and demo commands for Nyan animation.
- Introduce configuration options for animation speed, reduced motion, auto reveal, and automation toggles.
- Hook into VS Code tasks/debug sessions and expose a public `withProgress` helper API.
- Carry out initial accessibility pass (ARIA states, reduced-motion support, focus styling) for the Nyan panel.
